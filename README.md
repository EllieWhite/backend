**backend**

В среде исполнения Node.js нет браузера, поэтому document, window не будут использованы.
- В Node.js есть встроенная система модулей - можно создавать другие файлы, подключать и кк-то связывать их. c помощью функции require: require('./module');
- Больше доступа к различным сущностям, например, можем вывести глобальные переменные, пути;
- process.argv - объект, помогает считывать передаваемые параметры в node.js программу; второй параметр попадает в выводимый массив;

**Как работают модули**

    Есть 2 основных типа модулей Node.js:
        1. Встроенные модули в саму программу. Можно обратиться к модулю из любых программ, из любого места. 
        2. Сторонние модули, который устанавливаются с npm.

    Для разработки программы установим модули:
    npm i yargs - для чтения параметров и документации
